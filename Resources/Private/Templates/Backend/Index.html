In order to import images for Locations the Filenames in that directory need to begin with the "id" specified in the CSV followed by a ".".
Anything after that "." can be named as needed. The images will be imported alphabetically. For Example:
  • 1.Musterfirma.jpg
  • 1.Musterfirma Bild 2.jpg
  • 2.eine andere Firma.jpg

Example CSV 

id; name; contact; street; zip; city; country; region; phone; fax; url; latitude; longitude
